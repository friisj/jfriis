# Pulse - Definitions

> Glossary of terms specific to this project. Maintain as concepts evolve.

---

## Core Terms

| Term | Definition | Example |
|------|-----------|---------|
| Context Source | An external corporate tool from which Pulse ingests content | Slack workspace, Notion wiki, Linear project |
| Pulse Quiz | A short, generated quiz delivered on a recurring cadence | Weekly 5-question quiz on recent Slack announcements |
| Reinforcement Track | A personal spaced-repetition sequence targeting knowledge gaps | An employee's ongoing review of onboarding material |
| Knowledge Health | Aggregate measure of how well a team retains key information | 78% team accuracy on Q1 strategy questions |

---

## Related Concepts

| Concept | Relationship to This Project |
|---------|------------------------------|
| Spaced Repetition | Core learning science principle — quiz timing follows forgetting curves |
| Active Recall | Retrieving information strengthens memory more than re-reading |
| RAG (Retrieval-Augmented Generation) | Likely architecture for generating questions from corporate content |
