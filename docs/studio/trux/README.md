# Trux

> Side-scrolling monster truck survival game - A monochrome reimagining of Excite Bike

## Status

- **Phase:** Prototype
- **Temperature:** Warm
- **Started:** 2025-12-27

## Overview

Trux is a browser-based side-scrolling game where players pilot a parametrically customizable monster truck through procedurally generated terrain. Inspired by Excite Bike's physics-based gameplay, Trux emphasizes vehicle tuning and survival time over racing.

## Key Features

- **Monochrome line art aesthetic** - Clean, performant rendering
- **Procedural terrain generation** - Infinite, algorithm-driven levels
- **Parametric vehicle customization** - Adjust wheel diameter, suspension, weight, power
- **Physics-based gameplay** - Realistic momentum, gravity, collision
- **Survival scoring** - How long can you keep the truck upright?
